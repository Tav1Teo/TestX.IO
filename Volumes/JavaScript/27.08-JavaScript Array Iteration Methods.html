Array.indexOf()

Search an array for an element value and returns its position.

Note: The first item has position 0, the second item has position 1, and so on.

Syntax

array.indexOf(item, start)
item Required. The item to search for.
start Optional. Where to start the search. Negative values will start at the given position counting from the end, and search to the end.

Array.indexOf() returns -1 if the item is not found. If the item is present more than once, it returns the position of the first occurence.

Array.lastIndexOf()

Array.lastIndexOf() is the same as Array.indexOf(), but searches from the end of the array.
